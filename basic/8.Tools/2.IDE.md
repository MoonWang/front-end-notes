# VS Code

## 插件

- Chinese (Simplified) Language Pack for Visual Studio Code 中文包
- Beautify 代码格式化
- favorites 收藏夹(便于大型项目、node_module 快速定位)
- ESLint 代码规范检查
- GitLens 查看具体行代码 commit 信息
- Code Runner 直接执行文件
- Vetur vue 开发
- Open-In-Browser 直接在浏览器打开文件
- vscode-icons 显示文件图标(美化)
- Debugger for Chrome 用静态页面打断点调试
